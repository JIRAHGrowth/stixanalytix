01 · Reference

The four keys you'll actually use.

These cover ~95% of a review session. Learn them first; everything else below is for power users and bulk operations.

>
★ Jump to the next event in time, across goals/saves/distribution combined. The video-following key.
y
★ Keep the focused event (it was real).
n
★ Reject the focused event (false positive).
?
★ Open the full shortcuts overlay if you forget a key.

Navigation

>
Next event in time (across all sections)
<
Previous event in time
↓ or j
Next event in current section only
↑ or k
Previous event in current section
Tab
Switch to next section (goals → saves → distribution)
Shift+Tab
Switch to previous section

Use > when you're following the match video — it jumps to whatever happened next in time. Use ↓ when scanning within one category (e.g., reviewing all distributions back-to-back).

Mark the focused event

y
Keep (event was real)
n or x
Reject (false positive — Gemini was wrong)
Space
Toggle keep / reject

Bulk actions (current section only)

Shift+A
Accept all events in this section
Shift+R
Reject all events in this section
Shift+H
Accept all high-confidence events
Shift+L
Reject all low-confidence events
?
Show or hide the shortcuts overlay
Esc
Close overlay, or step out of a text field to re-enable shortcuts
How to practice (5 minutes) 1. Open any published match's review page.
2. Find the focus glow. Press > a few times — watch it jump across sections in time order.
3. Press y on one event, n on another. Watch the checkboxes tick.
4. Press ? to see the full overlay if you forget a key.
5. When typing in notes, shortcuts auto-pause. Esc to step out and re-enable.

The goal: drive one match end-to-end with only the keyboard. After one match, the muscle memory is built.

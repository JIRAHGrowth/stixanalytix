# StixAnalytix Brand Quick Reference

Full guidelines: [Stixanalytix_brandguidelines.pdf](./Stixanalytix_brandguidelines.pdf) (v1.0, March 2026)

## Color Palette

| Token | Hex | Role |
|-------|-----|------|
| Pitch Black | `#0A0A08` | Primary background (dark). Warm near-black |
| Chalk | `#EDEAE1` | Primary text/frame on dark. Warm off-white |
| Terminal Green | `#10B981` | Signature accent. Wordmark, signals, active states. **Never as large fill** |
| Forest | `#047857` | Terminal Green alternative on light backgrounds (WCAG contrast) |
| Paper | `#F2EFE8` | Primary light background for print and documents |

### Color principles
- **Dark backgrounds are home** — designed for screens in dark coaching environments
- **Green is the signal** — marks what matters, never a large fill
- **Warm neutrals only** — no pure white or pure black

## Typography

**Outfit** — Display, headlines, UI labels, the wordmark
- 800 (mark/headlines), 700 (subheads), 400 (supporting UI)

**DM Sans** — Body, long-form, captions, data
- 500 (UI/emphasis), 400 (body), 300 italic (editorial/captions)

Both free via Google Fonts.

### Type scale
| Role | Size |
|------|------|
| Cover / Display | 32pt |
| Heading 1 | 22pt |
| Heading 2 | 15pt |
| Lead paragraph | 12pt |
| Body copy | 10pt |

## Wordmark
- Always **lowercase**: `stixanalytix`
- Logo = goal mouth (three strokes: two posts, one bar, UEFA proportions) with net grid that doubles as spreadsheet grid
- 80s terminal nostalgia is intentional, not accidental

## Voice & Tone

> "A confident coach who commands attention without shouting."

**We are:** Direct, Precise, Knowledgeable, Human
**We are not:** Salesy, Buzzwordy, Casual to a fault, Academic

### Write this, not that

| ✓ Write this | ✗ Not this |
|-------------|-----------|
| "Save percentage dropped from 78% to 71% over the last five matches." | "Performance metrics indicate a downward trajectory in shot-stopping efficiency." |
| "The goalkeeper let in 3 goals from crosses in the last 4 games." | "Cross-related concession data shows concerning trends in aerial defensive performance." |
| "Distribution accuracy: 74%. Your target is 80%." | "Based on our analytics suite, we have identified opportunities for distributional improvement." |

## Use for all documents
Any document going forward (HTML, PDF, web, print) should honor:
- Color tokens above
- Outfit + DM Sans typography
- Voice & tone
- Lowercase wordmark

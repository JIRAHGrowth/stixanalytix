StixAnalytix — Review Keyboard Shortcuts

Tape this beside your keyboard. The four starred keys do 95% of the work.

The 4 keys you'll actually use

> ★ Jump to next event in time, across all sections. The video-following key.
y ★ Keep the focused event (it was real)
n ★ Reject the focused event (false positive)
? ★ Pop up the full shortcuts overlay

Navigation

> Next event in time (across all sections)
< Previous event in time
↓ or j Next event in current section
↑ or k Previous event in current section
Tab Switch to next section (goals → saves → distribution)
⇧+Tab Switch to previous section

Mark the focused event

y Keep (the event was real)
n or x Reject (false positive — Gemini was wrong)
Space Toggle keep / reject

Bulk actions (apply to the entire current section)

⇧+A Accept all events in this section
⇧+R Reject all events in this section
⇧+H Accept all high-confidence events
⇧+L Reject all low-confidence events

Misc

? Show / hide shortcuts overlay
Esc Close overlay, or step out of a text field to re-enable shortcuts
How to practice (5 minutes):
1. Open any published match's review page.
2. Find the focus glow (the highlighted event). Press > a few times — watch it jump across sections in time order.
3. Press y on one, n on another. Watch the checkboxes tick on / off.
4. Press ? to see the full overlay any time you forget a key.
5. When typing in a notes field, shortcuts pause automatically. Press Esc to step out and re-enable them.

The goal: review one match using only the keyboard. The four starred keys above are 95% of what you'll actually press.
